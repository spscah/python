An introductory python course

This course takes you on a week by week course through learning python.

Each week should has a focus, with questions for discussion in class and then to spend some time working at home.

This course is heavily influenced by How to Think Like a Computer Scientist, although some of the exercises are very different.

  1. Installing and Runing Python
  2. The Command Line
  3. Variables, Data Types and simple input/output
  4. User Interface using Turtles
  5. Simple functions
  6. Conditional statements
  7. Chatterbox Functions
  8. Iteration
